🚀 Quick Start Guide

5분 만에 구축하기

1. Splunk 설정

• Settings → Data Inputs → HTTP Event Collector

• 새 토큰 생성 및 fortimanager 인덱스 설정


2. XML 대시보드 생성

• Settings → User Interface → Views → New View

• 위의 XML 코드 전체 복사하여 붙여넣기


3. FortiManager 연동

• Policy Lookup API 활성화

• HEC로 이벤트 데이터 전송 설정


4. 실시간 모니터링 시작!

• 대시보드에서 실시간 보안 현황 확인
